Protocol
How It Works

Three steps from silence to the perfect track — audience driven, DJ controlled.

01 / SEARCH
🔍
Audience Requests
Audience joins with just their name. Search any song — Bollywood, international, anything on Deezer. Request it live. If it's already on the list, vote it up instead.
02 / VOTE
📊
Crowd Votes
Every song on the queue shows live interest and disinterest counts. The DJ sees exactly what the crowd wants — helping prioritize the most anticipated tracks.
03 / PLAY
🎚️
DJ Decides
The DJ picks what plays, reorders the queue, marks songs as Playing or Played. A live display screen keeps the whole room in sync — no shouting over the speakers.

Capabilities
Built for Real Events

Every feature exists because a real DJ-night scenario demanded it.

🔥 Live Realtime Sync
Firebase Firestore pushes every queue update, vote, and status change instantly to all screens — no refresh needed.
🎶 Deezer Search API
Search millions of tracks including recent Bollywood hits, international charts, and classics — all powered by Deezer via RapidAPI's free tier.
🗳️ Smart Voting
Each audience member gets one vote per song. Searching a song already on the queue auto-shows the upvote option instead of duplicating it.
🎪 Multi-Event Support
Every DJ can run separate events independently. Audience members pick which event to join — perfect for multi-room venues or different nights.

Initialization Sequence
Get Running in 15 min

No servers. No monthly bills. Just paste two keys and you're live.

01
Create Firebase Project
Go to console.firebase.google.com → New Project → Add Web App → copy your config object. Enable Firestore in test mode. Free forever on Spark plan.
🔗 console.firebase.google.com → Spark Plan (free)
02
Get Deezer API Key (RapidAPI)
Sign up at rapidapi.com → search "Deezer" → subscribe to the free plan (500 searches/day). Copy your X-RapidAPI-Key.
🔗 rapidapi.com → Deezer API → Free Plan
03
Paste Keys into the 3 HTML files
Open audience.html, dj.html, and display.html. Find the YOUR_API_KEY block near the top of each <script> tag and replace with your Firebase config + RapidAPI key.
04
Deploy to Vercel (free)
Push all 4 files to a GitHub repo. Connect the repo on vercel.com. Vercel deploys static HTML instantly for free with a public URL you can share.
🔗 vercel.com/new → Import GitHub Repo → Deploy
